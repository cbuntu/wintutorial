# Windows应用指南

## 概述
本指南为cbuntu.github.io系列教程指南中的Windows系列，指南中涉及了Windows系统的简介、安装及使用，为学习计算机的基础教程。

## 声明
本指南中所使用的系统安装镜像及软件均以学习为目的，不提供任何激活方法；您可选择正版或相关开源系统或软件。
