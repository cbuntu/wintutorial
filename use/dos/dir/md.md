# 目录类操作命令

## md命令说明及实例

### md说明
1. 功能：创建新的子目录
2. 类型：内部命令
3. 格式：md [盘符:][路径名]<子目录名>
4. 别名：mkdir
5. 使用说明：
	* 盘符：要建立子目录所在的驱动器字母，若省略即为当前驱动器
	* 路径名：要建立子目录的上级目录名，缺省则建在当前目录下

### md实例

```
**[terminal]
# 当前目录下建立子目录：cmd，再dir查看
C:\> md cmd
C:\> dir
cmd
```

```
**[terminal]
# 在cmd目录下建立子目录：code，再dir查看
C:\> md cmd\code
C:\> dir cmd
.        ..        code
```
