# 磁盘类操作命令

## label、vol命令说明及实例

### label说明
1. 功能：创建、更改或删除磁盘的卷标
2. 类型：外部命令
3. 格式：label [drive:][label]
4. 使用说明：
	* drive: 指定驱动器名，缺省为当前驱动器
	* label  指定要建立的卷标名，如缺省系统提示键入卷标名或删除原有卷标名
	* 卷标名由1到11个字符组成

### vol说明
1. 功能：显示磁盘卷标和序列号（如果存在）
2. 类型：外部命令
3. 格式：vol [drive:]
4. 使用说明：
	* drive: 指定驱动器名，缺省为当前驱动器

### label、vol实例

```
**[terminal]
# 为D盘建立卷标
C:\> label d: music
C:\>
```

```
**[terminal]
# 查看D盘卷标
C:\> vol d: 
驱动器 D 中的卷标是 music
卷的序列号是 5066-7A99
C:\>
```

```
**[terminal]
# 删除卷标
C:\> label d:
驱动器 D: 中的卷是 music
卷序列号为 5066-7A99
卷标(如果没有，请按 ENTER)?

是否删除当前的卷标(Y/N)? Y
C:\>
```
