# 其他类操作命令

## title命令说明及实例

### title说明
1. 功能：设置命令提示窗口的窗口标题
2. 格式：title [string]
3. 使用说明：
	* string 指定命令窗口的标题

### title实例
```
**[terminal]
C:\WINDOWS\system32\cmd.exe
**[path D:\]**[delimiter > ]**[command title demo]
```

```
**[terminal]
demo
**[path D:\]**[delimiter > ]**[command title demo]

**[path D:\]**[delimiter > ]
```
