# Windows XP简介

## 概述
* Windows XP是Microsoft在2001年10月25日推出的基于X86、X64架构的PC和平板电脑的操作系统，包括商用及家用的台式电脑等，大小为575MB~1G。

* 其名字“XP”的意思是英文中的“体验(Experience)”，是继Windows 2000及Windows ME之后的下一代Windows操作系统，也是微软首个面向且使用Windows NT 5.1架构的操作系统。

## 用户界面
* 桌面：Windows XP拥有一个叫做Luna（月神）的用户图形界面，视窗标志改为较清晰亮丽的四色窗标志。窗口增加了工具条。

* 任务管理器：XP系统相对Windows 2000对任务管理器改动不是很大，只是更加符合Luna界面效果；在XP系统下按Ctrl + Alt + Delete进入安全界面，间接打开任务管理器，是XP系统的一大特色。

* 开始菜单：Windows XP率先使用双列菜单，为之后的Windows版本的开始菜单提供了基础。

* 资源管理器
Windows XP的文件资源管理器使用了全彩界面。相比上一个版本改变不大。但也适应了其Luna界面。

## 主要版本
* 家庭版(Windows XP Home Edition)

是面向家庭用户的版本。主要表现为：没有组策略、远程桌面、EFS文件加密、多语言、连接Netware服务器的功能，只支持1个CPU和1个显示器，不具备访问控制和IIS服务以及不能归为域等。

* 专业版(Windows XP Professional)

是面向企业、开发人员的版本，与Home Edition相比提供更加全面的功能，是Windows XP的全功能版本。Windows XP Professional Edition支持双CPU系统。
