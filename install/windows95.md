# Windows 95安装

## 新建Windows 95虚拟机
> 1. 复制msdos7.1 -> windows 95 -> 完全复制
> 2. 设置 -> 类型：Microsoft Windows -> 版本：Windows 95
> 3. 设置 -> 光驱加载：Win95osr20.iso

## Windows 95 安装
